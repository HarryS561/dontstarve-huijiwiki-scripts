# Recipes
本模块用于解析并上传`scripts/recipes.lua`中的配方到维基`Data:DSTRecipes.tabx`。

## 使用

1. 按照根目录下的README.md中的步骤配置好环境、账号信息和游戏目录。
2. 切换工作目录到`Recipes`目录下，
3. 执行run.py文件

正常情况下输出类似于:
```
成功解析 897 个配方
新添加了41个配方
删除了1个配方
修改了4个配方
是否确认更新? (y/n)
```
之后输入`y`确认更新，即可上传到维基`Data:DSTRecipes.tabx`。
